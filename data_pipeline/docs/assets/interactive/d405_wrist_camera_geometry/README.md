# D405 Wrist Camera Geometry Viewer

This folder contains a standalone HTML/Three.js viewer for the D405 wrist-camera
mount geometry.

Open from the MkDocs page:

```text
data_pipeline/docs/d405-wrist-camera-mount.md
```

or directly:

```text
data_pipeline/docs/assets/interactive/d405_wrist_camera_geometry/wrist_camera_geometry_gui.html
```

The viewer is self-contained except for the local Three.js module in `vendor/`.
